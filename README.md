## Blog build with React and NodeJs

### Pre

NodeJs, Mongdb

### Start

Project clone:

```bash
git clone git@github.com:ivanberry/Amy.git
```

Dependecy install:

```bash
cd server
npm install

cd client
npm install
```

### Run tests

Run tests seperately:

```bash
cd server
npm test
```
